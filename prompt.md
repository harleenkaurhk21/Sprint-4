# AI Prompts Used

## Prompt 1
Create a responsive AI Cover Letter Generator using HTML, CSS, and JavaScript with a clean and modern user interface.

## Prompt 2
Build a Node.js and Express.js backend that integrates with the Google Gemini API to generate professional cover letters.

## Prompt 3
Generate a professional cover letter based on the candidate's name, job role, company name, skills, and resume details. The letter should be 250–350 words, formal, personalized, and end with the candidate's name.

## Prompt 4
Implement PDF resume upload using PDF.js and extract text automatically to include it in the AI prompt.

## Prompt 5
Add features including Dark Mode, Local Storage, Character Counter, Form Validation, Copy to Clipboard, and Download as PDF using jsPDF.

## Prompt 6
Configure the application for deployment on Render using environment variables to securely store the Gemini API key.

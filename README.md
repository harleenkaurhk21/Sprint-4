# AI Cover Letter Generator

An AI-powered web application that generates professional and personalized cover letters using the Google Gemini API. Users can enter their details, upload a resume in PDF format, and instantly receive a customized cover letter.

## 🚀 Live Demo

https://sprint-4-5bku.onrender.com

## 📌 Features

- Generate AI-powered cover letters
- Upload resume in PDF format
- Extract text from PDF automatically
- Responsive user interface
- Dark mode support
- Character counter
- Form validation
- Copy generated cover letter
- Download cover letter as PDF
- Local storage support

## 🛠️ Technologies Used

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js

### API
- Google Gemini API

### Libraries
- PDF.js
- jsPDF
- CORS
